# 行情报价采集 — 功能汇报

---

## 一、功能范围

| 模块 | 功能点 | 说明 |
|------|--------|------|
| 行情面板 | 规格成交价 | 按品类/规格展示当日市场价，支持查看价格区间和成交走势 |
| | 成本因素 | 冰衣、运费、关税等成本拆解，辅助理解价格构成 |
| | 行业资讯 | 虾产业动态、政策变化、供需分析 |
| | 交易市场 | 挂牌/拍卖货源公开交易，吨数为主件数辅助 |
| 价格采集 | 用户报价提交 | 船东/贸易商/业务员提交某规格的当日报价 |
| | 信息来源分类 | 实际成交、市场询价、买方报价、卖方报价、行业判断 |
| | 可信度评估 | 基于报价偏离度+来源权重计算，实时反馈可信度评分 |
| | 交换机制 | 未提交报价时，报价人数/可信度/深度资讯锁定，提交后解锁 |
| 激励体系 | 等级徽章 | 新手→达人(3次/天)→专家(5次/天)，等级越高可信度加成越高 |
| | 连续奖励 | 连续3天+2可信度，连续7天+5可信度 |
| | 报价记录 | 查看历史提交记录、当日报价次数进度 |

---

## 二、核心价值

**对船东：**
- 掌握当日真实行情，定价有依据，不盲卖
- 品控报告+行情数据双重背书，议价更有底气

**对平台：**
- 用信息换信息：用户想看别人的报价，自己必须先报——越多人参与，数据越真实
- 报价数据沉淀后形成行业价格指数，是平台的核心数据资产
- 用户从"被动看行情"变成"主动贡献行情"，日活和粘性双提升

**对买家/贸易商：**
- 可信度标识降低信息不对称风险
- 多来源交叉验证（成交/询价/买卖方报价），价格参考更全面

---

## 三、业务逻辑

用户进入行情页后：

**未提报状态：**
- 报价人数模糊锁定
- 可信度隐藏锁定
- 部分资讯锁定
- 引导"提交报价解锁行情详情"

**点击报价流程：**
1. 选择品类 → 选择规格 → 输入价格 → 选择来源（实际成交/市场询价/买方报价/卖方报价/行业判断）
2. 系统实时计算可信度：
   - 报价与市场均价偏差 ≤3% → 高可信 90-99分
   - 偏差 3%-8% → 中等可信 70-90分
   - 偏差 >8% → 低可信 30-70分
   - 来源加成：实际成交 +5分
3. 提交后：
   - 解锁全部行情详情（人数、可信度、资讯）
   - 等级进度更新（新手→达人→专家）
   - 激励继续报价（可信度加成、连续奖励）

**可信度算法核心：** 同一规格多人报价时，偏离众数越近可信度越高，来源为"实际成交"额外加权。参与人数越多，可信度参考价值越高。

**交换机制核心：** 不报价就看不到别人的报价信息——用你的信息换大家的信息，参与越多解锁越多，形成正向飞轮。
